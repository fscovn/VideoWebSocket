# VideoWebSocket
WebSocket Server for video
# Yêu cầu:
Websocket chạy trên nền tảng linux (ubuntu)
## Cài đặt thư viện:
- Cài đặt opencv: 
sudo apt install libopencv-dev
- Cài đặt websocketpp: 
sudo apt install libwebsocketpp-dev

## IDE
Cài đặt netbeans https://netbeans.apache.org/download/index.html
Active plugin C/C++
